Import Tile

This window displays an image and allows you to select a tile-sized portion of it to import into the full screen Tileset editor.

The title bar of this window displays the pixel coordinates. The size of the rectangle is determined by the size of the tiles in the tileset being imported into. When you click, the full screen tileset editor will come up with the selected rectangle in the grid. It will not actually be saved anywhere in the tileset until/unless you click the store button.
