Technical Support and Contact Information

The Scrolling Game Development Kit has a web site at http://gamedev.sourceforge.net/. It is also an open source project maintained on the web at http://www.sourceforge.net/projects/gamedev/ (the same site that hosts the main web site). The open source web site has places to report bugs, ask questions, download the latest code and so forth. If you are unable to contact the web site, the author can be contacted via e-mail directly at BlueMonkMN@email.com.

If you would like to contact the author of The Scrolling Game Development Kit for any other reason, feel free to e-mail at BlueMonkMN@email.com. This is my address and I was responsible for all aspects of GameDev: the design and development of GameDev, supporting DLL's (BMDXCtls.dll and ScrHost.dll), the tutorial, all documentation (as of this writing) and the content of the gamedev.sourceforge.net web site.

For support on VBScript, see http://msdn.microsoft.com/scripting/. Scripting support in GameDev is not currently fully documented, but I will be happy to answer specific questions at the open source web site, or if neccessary, directly to my e-mail address. Using the object viewer in Visual Basic, looking at the last step in the tutorial and viewing the source code are the only ways to learn about scripting support at the moment.
